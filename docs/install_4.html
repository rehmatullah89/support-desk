ReCaptcha Service

  • Installation /
  • ReCaptcha Service
For spam prevention on the ticket creation page, Maian Support utilises the reCaptcha service from Google, a free anti-bot prevention solution. For more information, click here. If you aren`t sure about spam or its terminology, click here. reCaptcha offers a lot of flexibility with its code, so see below for more information.

Enable in Maian Support

In the admin menu go to "Settings & Tools > Settings". Select the "Other Options > reCaptcha" tab.



Create a free account and refer to the options for the reCaptcha settings here.

Parameter Settings

Certain parameter settings, such as language and themes are found in the following file: admin/control/recaptcha.php
